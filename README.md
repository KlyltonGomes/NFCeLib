🔗 **Uma biblioteca Java poderosa para emissão de NFC-e (Nota Fiscal do Consumidor Eletrônica), simplificando a integração com a SEFAZ.**  

## ✨ **Recursos**  
✅ Emissão de NFC-e sem complicações  
✅ Assinatura digital dos documentos fiscais  
✅ Validação XML antes do envio  
✅ Simulação de transmissão para testes (futuro)  
✅ Geração de DANFE em PDF (futuro)  

---

Observações:
Fácil integração com projetos Spring.
Configuração própria para evitar conflitos com Beans.
Módulos desacoplados para personalização.
